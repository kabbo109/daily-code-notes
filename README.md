# Daily Code Notes

This is a very easy project for beginners.

You can use this repository to:
- Write simple code notes
- Track daily learning
- Practice editing files
- Build a habit of GitHub commits

How to use:
1. Open any file
2. Add a small note or change
3. Save the file
4. Commit your update

Goal:
Learn something small every day.

Notes:
Write your daily code notes below.
